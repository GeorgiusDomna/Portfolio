## Run Locally

clone the project

```bash
  git clone git@github.com:GeorgiusDomna/Portfolio.git
```

go to the project directory

```bash
  cd Portfolio
```

install dependencies

```bash
  npm install
```

start the server

```bash
  npm run dev
```
